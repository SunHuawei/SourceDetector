Source Detector features

Source Detector helps security researchers and frontend teams discover source maps, scan for exposed frontend secrets, review client-side exposure signals, and export evidence for validation.

Source map discovery

Detect source map files referenced by live pages and related client-side assets while browsing. This helps you find exposed source bundles faster and review what is reachable from production pages.

Frontend secret scanning

Scan bundled assets with built-in and custom rules to surface suspicious patterns, including exposed API keys, AI-related tokens, and other client-side secret leakage clues.

Client-side exposure review

Inspect findings in a structured explorer so you can review evidence, understand context, and distinguish between a public configuration clue and a higher-risk secret exposure candidate.

Evidence export

Export selected artifacts as ZIP bundles for offline validation, reporting, or internal handoff. This makes it easier to preserve exact evidence without rebuilding the same browsing session.

Who these features are for

  • Bug bounty hunters doing browser-based reconnaissance
  • Security researchers reviewing client-side attack surface
  • Frontend teams auditing exposed production assets
  • Developers verifying what is publicly reachable after deployment
