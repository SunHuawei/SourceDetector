Client-side exposure scanner for public web assets

Source Detector helps you review what a public website reveals through source maps, shipped assets, suspicious strings, and other client-side exposure clues.

What client-side exposure review means

Client-side exposure review is the process of understanding what the browser can already see: source maps, config fragments, identifiers, public endpoints, and suspicious strings embedded in shipped code.

How Source Detector helps

  • Discovers source map references
  • Collects browser-side evidence
  • Scans assets for suspicious patterns
  • Exports artifacts for validation and reporting
