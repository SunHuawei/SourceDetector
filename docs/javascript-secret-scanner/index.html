JavaScript secret scanner for production bundle review

Source Detector helps you inspect production JavaScript bundles for exposed keys, suspicious tokens, and risky secret-like strings that deserve manual validation.

Why scan JavaScript bundles

Modern websites often expose large client-side bundles that contain integration logic, config fragments, and other implementation details. A JavaScript secret scanner helps you review those assets faster and with more structure.

What you can review

  • Possible hardcoded API keys
  • Suspicious token-like strings
  • Public config clues worth deeper review
  • Patterns surfaced by built-in and custom rules
