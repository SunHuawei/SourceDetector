## Asset specifications

- Screenshot format: PNG (sRGB), no visible private data.
- Screenshot size:
  - Primary: 1600x1000
  - Backup: 1280x800
- Video format: MP4 (H.264), 1080p preferred, 24-30fps.
- Video length: keep each clip under 60 seconds.
- UI language: keep one English baseline capture per workflow; localized variants are optional.
- Redaction: hide sensitive domain names, keys, tokens, emails, and internal identifiers.

## Review checklist before handoff

- Verify all links and labels in captures match current UI.
- Ensure no real credentials or confidential source code appear.
- Keep consistent browser theme and zoom (100%) across captures.
- Confirm filenames follow this document exactly for batch delivery.
