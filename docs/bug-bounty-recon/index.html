Bug bounty recon browser extension for frontend asset review

Source Detector helps bug bounty hunters move faster by discovering source maps, scanning JavaScript bundles, and preserving client-side evidence during browser-based reconnaissance.

Why it fits bug bounty recon

  • Works in the browser where recon already happens
  • Helps surface source maps and suspicious asset patterns quickly
  • Preserves evidence for later validation
  • Supports manual judgment instead of hiding context

Useful recon scenarios

Source Detector is useful when reviewing production JavaScript, checking whether source maps are exposed, and spotting strings or configuration hints that may justify deeper testing.

Scope boundary

It is a focused recon and evidence workflow, not a full attack automation platform. It helps you see more of the client-side surface, but exploitation decisions remain manual.
